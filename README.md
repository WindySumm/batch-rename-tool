# 批量重命名小应用

基于 Python **Tkinter** 的 Windows 友好批量重命名工具。规则可组合预览，执行后支持**仅本次程序运行期间**的撤销（关闭窗口后记录清空）。

## 环境要求

- Windows 10/11（主要使用场景）
- [Python 3.10+](https://www.python.org/downloads/)（安装时勾选 **tcl/tk** 以启用 tkinter）

依赖见 `requirements.txt`（当前无第三方包）。

## 运行方式

在dist目录执行：

```
批量重命名.exe
```

## 功能说明

界面按步骤组织：**先选文件**，再在标签页里选规则，**大半屏为「改完后的预览」**（绿色表示会变，灰色表示与现在相同，红色为错误提示）。

| 模块 | 说明 |
|------|------|
| 选文件 | 选择文件夹；可选「包含子文件夹」；可按扩展名筛选（留空 = 全部） |
| 一键整理名字 | 下拉选择常见整理方式。选「文本替换」时，会在**本页说明文字下方**展开输入区，并**自动向上滚动**规则区以便看到输入框；选其它方式时该区域隐藏 |
| 加前后缀 / 序号 | 在主文件名前/后加字；可选按列表顺序自动编号 |
| 预览 | 实时显示「现在的名字 → 改完之后」；顶部有一行统计说明 |
| 执行 / 撤销 | 仅无错误时允许执行；**仅本次运行内**可撤销上一批 |

### 规则应用顺序

对每条文件依次：

1. **一键预设或高级替换**（预设大多只改「主文件名」，扩展名相关项会单独处理扩展名）  
2. 拆分主名与扩展名后：**序号**（若勾选）→ **前缀** → **后缀**  

预设列表与说明文字在代码中集中在 `rename_core.PRESET_CHOICES`，便于以后继续加「常见场景」。

### 安全与限制

- 重命名采用**两阶段临时名**，减少互斥改名导致的覆盖问题。
- 会检测：非法 Windows 文件名字符、批内目标重名、与磁盘中已有文件的冲突。
- **撤销**依赖改名后路径仍存在；若你在外部移动/删除了文件，撤销可能失败。
