# 更新日志

本文档记录项目的**用户可见变更**与简要技术说明，便于后续迭代与发布说明。  
建议格式：按版本倒序，新版本写在最上方。

---

## [0.2.2] - 2026-04-03

### 新增

- 预览窗口最左侧增加「序号」列，显示文件的顺序

### 变更

- 窗口最小宽度调整为810像素，确保所有文字完全显示
- 使用 PyInstaller 打包为单文件可执行程序：`批量重命名.exe`
- 打包参数：`--onefile --noconsole --icon=Re_icon.ico`（无控制台窗口，带自定义图标）

### 移除

- 移除了拖动排序功能，保留序号显示

## [0.2.1] - 2026-04-03

### 变更

- 「高级：自己替换文字」**仅在选择「（高级）自己替换几个字…」时**在「一键整理名字」里、说明文字**下方原位置**自动展开；改选其它预设时**整块隐藏**。
- 选中高级并展开后，**自动滚动**上方规则区，使「自己替换文字」尽量出现在可视区域内；采用 `Map` 事件 + 多次延后执行，并用「屏幕坐标差 / 父链 winfo_y」双路计算偏移，减少滚动失效。
- 上方「选文件 + 规则」整体放入可纵向滚动区域（右侧滚动条保留，不再显示顶部操作提示文案）。

## [0.2.0] - 2026-04-03

### 新增

- **一键整理名字**：下拉框内置多种常见改名场景（空格/下划线、去掉复制后缀 `(1)`、扩展名小写等），用自然语言说明；底层可用正则，界面不展示公式。
- **（高级）自己替换几个字**：仍支持原来的「查找 → 替换」与区分大小写，仅在选中该预设时显示输入区。
- **强化预览区**：上下分栏，下方大区域为「改完后的文件名预览」；表格列标题改为更直白的「现在的名字 / 改完之后会变成」；颜色区分将改名 / 不变 / 错误；预览上方增加一行统计说明。

### 变更

- 配置项 `RenameRuleConfig` 增加 `replace_mode`（预设 id），与旧版「只有查找替换」相比，默认改为「不额外整理」。
- 主窗口标题与按钮文案偏新手向表述。

### 说明

- 预设 id 与文案维护位置：`rename_core.PRESET_CHOICES` 与 `apply_text_stage()`。

## [0.1.0] - 2026-04-03

### 新增

- 基于 Tkinter 的图形界面：选择目录、子目录开关、扩展名过滤。
- 规则：查找替换（可选区分大小写）、主文件名前缀/后缀、可选序号（起始、步长、位数、分隔符、名前/名后）。
- 预览列表：新文件名预览、非法名/批内重名/磁盘冲突提示。
- 批量执行重命名（两阶段临时文件策略降低覆盖风险）。
- **仅本次运行内**按批次撤销上一组成功操作（`UndoSession`，不写入磁盘）。

### 说明

- 未包含 exe 封装；依赖标准库，建议 Python 3.10+。
- 已知局限：无规则预设持久化、无跨会话撤销。（0.2.0 起内置「一键预设」，高级用户仍可用自定义替换。）
